## Beskrivning
Här skriver du en mer djupgående text om ditt projekt. Förklara vad det gör, varför det skapades
och ge en övergripande bild av hur det fungerar.
Vårt projekt är ett webb-baserat quiz-spel som påminner om Kahoot men den är enkelt att följa och har mer avancerade funktioner. Vi skapade det för att göra pluggandet roligare och mer interaktivt. Spelet har inbyggda färdiga kurser i webbutveckling och matematik i tre olika svårighetsgrader, men det bästa är att användaren kan skapa helt egna quiz. Man kan lägga till upp till fyra svarsalternativ, välja vilket som är rätt, och till och med klistra in en bildlänk till frågan. Allt sparas automatiskt på datorn med hjälp av webbläsarens inbyggda minne (LocalStorage). Vi har också byggt ett Score-system som sparar poängrekord, och en smart delningsfunktion där man kan kopiera sitt quiz som en hemlig kod och skicka till en kompis. Som en rolig detalj finns det även en interaktiv robot på skärmen som reagerar och ger feedback när man spelar.

## Kom igång
### Förutsättningar
För att kunna använda programmet krävs vissa grundläggande verktyg och programvara. Dessutom för koden behöver du ladda ner Visual Studio Code.  Programmet är byggt med HTML, CSS och JavaScript, vilket innebär att ingen extra installation av programmeringsspråk behövs. En webbläsare, till exempel Google Chrome eller Microsoft Edge, krävs för att kunna öppna och testa webbplatsen. 
 

### Installation (om möjligt)
* Hur och var man laddar ner programmet.
Du laddar ner Visual Studio Code genom att söka på Google "Visual Studio Code 2026”. Du behöver inte göra någon ytterligare modifiering.

* Eventuella modifieringar som behöver göras i filer eller mappar (t.ex. miljövariabler).
Koden får du i Github. Det kan behöva spara och packa upp filerna i filer, vilka lägger du sen i en ny map.

### Köra programmet

Ladda ner Visual studio code
Hämta filer från github and packa upp de
Kör visual studio koden välj “open folder” 
Välj den mappen där du lagt filer
Nu är programmet redo för att köra
